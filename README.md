# Cookie Cats A/B 测试分析

基于手游 Cookie Cats 的 9 万用户 A/B 测试数据,分析关卡位置从第 30 关改为第 40 关对用户留存的影响。

## 业务背景

Cookie Cats 是一款手游,在游戏初期会设置一道"关卡墙"要求玩家等待或付费才能继续。产品团队想验证:**把关卡墙从 gate_30 推迟到 gate_40,是否能提升用户留存?**

## 数据说明

| 字段 | 说明 |
|------|------|
| userid | 用户 ID |
| version | 实验分组(gate_30 / gate_40) |
| sum_gamerounds | 安装后 14 天内的游戏轮数 |
| retention_1 | 1 日留存(True/False) |
| retention_7 | 7 日留存(True/False) |

样本量:gate_30 约 44,700 人,gate_40 约 45,500 人。

## 分析方法

从三个角度互补验证:

1. **卡方独立性检验**:判断两组留存率差异是否统计显著,配合 Cramér's V 评估效应量
2. **贝叶斯 A/B 测试**:基于 Beta-Binomial 共轭,给出 P(gate_30 > gate_40) 的直观概率
3. **KS 检验 + CDF 可视化**:比较两组游戏深度分布是否一致

## 核心结论

| 指标 | 方法 | 结果 |
|------|------|------|
| retention_1 | 卡方检验 | p=0.074, V=0.006(不显著) |
| retention_7 | 卡方检验 | p=0.002, V=0.011(显著) |
| retention_7 | 贝叶斯 A/B | **P(gate_30 > gate_40) = 99.91%**,提升 0.82pp,95% 可信区间 [0.31pp, 1.33pp] |
| sum_gamerounds | KS 检验 | 两组 CDF 几乎重合,游戏深度无差异 |

**结论**:gate_30 优于 gate_40,效应集中在 7 日留存。关卡位置不改变玩家游戏行为(轮数分布一致),仅影响用户是否持续回来。

**业务建议**:维持 gate_30,不推进 gate_40 改动。

## 方法论要点

- **大样本 + 小效应**:9 万样本下即使 0.82pp 的差异也能稳定检出 p<0.05。**必须结合效应量(V)判断实际意义**,不能只看 p 值
- **贝叶斯的业务价值**:P(A>B)=99.91% 比 p=0.002 直观得多,可信区间可直接用于决策
- **双派互相印证**:频率派(p 值)与贝叶斯结论一致(区间不跨零),增强结论可靠性
